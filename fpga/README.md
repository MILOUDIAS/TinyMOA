
# TinyMOA FPGA Support

This directory contains files and documentation for deploying the TinyMOA CPU on FPGA boards, however, you cannot run the analog CIM accelerator on an FPGA.

*This document is WIP*

## Contents
- Board constraint files (e.g., Alchitry Cu)
- FPGA-specific build instructions
- Notes for supported boards and pin mappings

Refer to [docs/Setup.md](../docs/Setup.md) for toolchain setup and [docs/ISA.md](../docs/ISA.md) for processor details.
