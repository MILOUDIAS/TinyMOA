## Scratchpad internal SRAM

IHP SG13G2 pre-generated 512x32 SRAM macro (`RM_IHPSG13_1P_512x32_c2_bm_bist` renamed to `SG13G2_512x32_SP`).

Setup is based on [Urish's SRAM test](https://github.com/urish/ttihp-sram-test/tree/main)

Pre-generated IHP SRAM macros are found [here](https://github.com/IHP-GmbH/IHP-Open-PDK/tree/main/ihp-sg13g2/libs.ref/sg13g2_sram).
